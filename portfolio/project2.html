Golden Catalog

  View Project   View Code

The Classified Ads Platform for University Students. This full stack application is built with firebase database and authentication. The server is hosted by Heroku. This website features a multi page React application with React Router. The purpose of this long-term project is to provide a more acessible and organized platform for students to find what they need. This project is still on going, with plans to ad an anonymous posting feature.
