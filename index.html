How I broke into the Web Development niche

Front-end and back-end validation. If you are dealing with production level code, you are going to have to do both of these. Front-end validation deals with constraining input values to show the user what they need to type in. Even back end validation is crucial because there might be edge cases or bugs in how we set up our front end validation or even worse, the user has turned off their JavaScript for the page.

Once code is submitted, the back-end checks the form and either sends the validated form to the database or sends back an error to the front end for the user to see.
